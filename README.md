# Requirements

1.  PHP >= 7.2.0
2.  Lumen 7.0
3.  Mysql

# Installation

1.  Clone the project : **git clone** Github repo
2.  Create database
3.  Copy .env.example to .env (If required you change the database credentials)
4.  Composer install
5.  php artisan migrate
6.  php artisan db:seed
